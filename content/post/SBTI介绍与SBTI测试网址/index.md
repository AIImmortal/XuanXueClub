## 一、什么是 SBTI？

**SBTI = Silly Big Personality Test**（超傻人格测试）

- **来源**：2026 年 4 月由 B 站 UP 主制作、模仿 MBTI 格式的**恶搞 / 娱乐测试**

- **性质**：**无心理学理论、无科学依据、无信效度**

- **核心**：**反映当下情绪、生活状态、自嘲感**，不是 “性格类型”

- **结果标签**（典型）：

    - **死者（DEAD）**：精神躺平、虚无感

    - **吗喽（MALOU）**：打工人摆烂、拒绝内卷

    - **送钱者（ATM\-er）**：总付出、老好人

    - **草者（FUCK）**：暴躁、爱吐槽、反规则

    - **愤世者（SHIT）**：嘴上骂、私下扛

    - **小丑（JOKE\-R）**：表面搞笑、内心委屈

- **特点**：多次测结果常变、**高度随机**

## 二、什么是 MBTI？

**MBTI = Myers–Briggs Type Indicator**（迈尔斯 \- 布里格斯类型指标）

- **来源**：1940 年代基于**荣格心理学**，由布里格斯母女开发

- **性质**：**专业性格测评工具**（虽学术有争议，但有理论 \+ 标准化量表）

- **4 大核心维度**：

    1. **精力来源**：E 外向 / I 内向

    2. **信息获取**：S 实感 / N 直觉

    3. **决策方式**：T 思考 / F 情感

    4. **生活方式**：J 判断 / P 知觉

- **结果**：**16 种稳定人格类型**（如 INTJ、ENFP、ISTJ）

- **用途**：自我认知、职业规划、团队沟通、婚恋理解

## 三、SBTI vs MBTI：核心区别

|维度|**MBTI**|**SBTI**|
|---|---|---|
|**全称**|Myers–Briggs Type Indicator|Silly Big Personality Test|
|**属性**|**专业性格测评**（有理论）|**纯娱乐 / 玩梗**（无理论）|
|**测什么**|**稳定的人格偏好**（你是谁）|**当下情绪 / 生活状态**（你现在咋样）|
|**理论基础**|荣格心理学、标准化量表|**无科学依据**、算法随机|
|**结果稳定性**|相对稳定（人格偏长期）|**极不稳定**（多次测不一样）|
|**结果风格**|中性 / 正向标签（建筑师、提倡者）|**自嘲 / 丧 / 暗黑 / 扎心**（死者、吗喽）|
|**目的**|自我理解、成长、职业参考|**解压、玩梗、社交吐槽、发圈**|
|**学术地位**|流行但有争议（企业常用）|**纯网络梗**，心理学界不认|

**一句话总结**：

- **MBTI 回答：你是什么样的人（长期性格）**

- **SBTI 回答：你现在过得怎么样（短期情绪）**

## 四、为什么 SBTI 突然火了？

1. **反精英、反鸡汤**：不用 “优秀人设”，直接承认**累、丧、摆烂、委屈**

2. **极强共鸣**：戳中年轻人学业 / 职场压力、**自嘲文化**

3. **社交货币**：结果好发朋友圈、好聊天、好自嘲

4. **MBTI 过气感**：宣传语 “MBTI 已过时，SBTI 来了” 自带话题

简单说：
想**认真了解自己性格** → 看 **MBTI**；
想**吐槽当下生活、解压玩梗** → 测 **SBTI**。
